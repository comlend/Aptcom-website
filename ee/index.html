A modern  maintenance system.

the problem

There is a broken window in the lobby - and it should be reported - but maybe someone else has already? And you don't want make the effort to phone the manager just to be told that - and you are late for work so.... maybe you just don't report it.

solution

From your mobile - you can see if it has been reported already, and what has been actioned if it has. If not - you can report it easily from wherever you are and have it logged in the system where it is sent  directly to the correct person, and where it can be seen by others, saving everyone time - and money.

A safer environment.

With better instant communication comes better safety. For example - there is a break in - you can post it on the news board which goes instantly to all users. Or smaller things - easy, efficient communication is key.
